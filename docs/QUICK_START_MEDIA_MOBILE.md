# ⚡ Quick Start: Media Library & Mobile Admin

## 🚀 AKSES CEPAT

### Media Library
```
URL: http://localhost:3000/admin/media
Menu: Sidebar → "Media Library"
```

### Mobile Admin
```
URL: http://localhost:3000/admin
Hamburger Menu: Top-left corner (mobile only)
```

---

## 📸 MEDIA LIBRARY - 30 DETIK

### Upload File (Drag & Drop)
```
1. Buka /admin/media
2. Drag file dari komputer
3. Drop ke area upload
4. Done! ✅
```

### Copy URL
```
1. Hover ke gambar
2. Klik icon download
3. URL copied! ✅
```

### Delete File
```
1. Centang checkbox
2. Klik "Hapus (X)"
3. Confirm
4. Done! ✅
```

## 🐛 TROUBLESHOOTING

### Media Library tidak muncul
```
1. Hard refresh: Ctrl + Shift + R
2. Check URL: /admin/media
3. Check sidebar menu
```

### Hamburger menu tidak muncul
```
1. Resize browser < 1024px
2. Or test di mobile device
3. Should appear top-left
```

### Upload gagal
```
1. Check file type (image only)
2. Check file size (< 5MB)
3. Check internet connection
```
