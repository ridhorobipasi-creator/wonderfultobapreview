fix: Slider tidak muncul di admin panel CMS Beranda Tour

MASALAH:
- Slider tidak tampil di /admin/tour-landing (hanya tombol "+ TAMBAH SLIDE")
- Padahal di homepage tampil 5 slider dengan sempurna
- Admin tidak bisa mengelola slider yang sudah ada

ROOT CAUSE:
- Type mismatch: Database menyimpan price sebagai string ('3500000')
- Component HeroSlider.tsx mengharapkan price sebagai number
- Interface SliderItem salah mendefinisikan price sebagai string

SOLUSI:
1. Update interface SliderItem (price: string → number)
2. Tambah auto-conversion di fetchContent() untuk backward compatibility
3. Buat handleSaveWithConversion() untuk ensure price selalu number
4. Tambah InputGroupNumber component dengan type="number"
5. Update input field dan template untuk slider baru
6. Buat migration script prisma/fix-slider-price.ts
7. Run migration untuk fix data di database
